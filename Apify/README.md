# Apify Projeleri

Apify MCP ve REST API üzerinden çalışan veri çekme projeleri.
Token: `.sovereign_env` → `APIFY_API_TOKEN`

## Projeler

| Proje | Klasör | Durum |
|-------|--------|-------|
| YouTube Kanal Video Listesi | `yt-channel-video-list/` | 🔄 Geliştiriliyor |

## Kullanım

```bash
source /root/.sovereign_env
python3 Apify/yt-channel-video-list/fetch_videos.py "@KanalAdi"
```
