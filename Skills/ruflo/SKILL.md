# Ruflo (Orkestrasyon)

Claude Code için geliştirilmiş en gelişmiş swarm yönetim platformudur.

## ⚙️ Kurulum ve Çalıştırma
Proje `/root/ruflo` dizinindedir.

```bash
cd /root/ruflo
npm install
npx ruflo init --wizard
```
