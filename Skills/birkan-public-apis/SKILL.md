# public-apis Skill

Bu skill, geliştiricilerin yeni projeleri için ücretsiz dış veri kaynakları veya API'ler aradığı durumlarda devreye girer.

## Ne Zaman Kullanılmalı?
- Kullanıcı "ücretsiz bir hava durumu API'si var mı?", "kripto verisi çekebileceğim bir API öner" gibi sorular sorduğunda.
- Hackathon veya hızlı prototip geliştirme aşamalarında dış veri kaynağına ihtiyaç duyulduğunda.
- Dummy (sahte) veri yerine gerçek veri kullanılmak istendiğinde.

## Nerede Bulunur?
Referans alınan tüm liste ve kategoriler, daha önce indirilen `/Users/birkan/Desktop/Work /00 Github PROJELERI/public-apis` klasöründe yer alan repo ve Readme dosyalarında mevcuttur.

## Kullanım:
Gerekli API'yi önermek için public-apis listesini tarayın, kategoriye göre uygun olanı seçip kullanıcıya iletin. Kullanım dokümanlarını ilgili API'nin kendi web sayfasından çekmesine yardımcı olun.
