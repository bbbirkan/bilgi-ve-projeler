# Glama MCP Server Dizini

Web tabanlı, aranabilir MCP kataloğu. Klonlanmaz — doğrudan tarayıcıdan kullanılır. punkpeye'nin awesome-mcp-servers repo'suyla senkronize, üstüne 22,500+ server içeriyor.

**URL:** https://glama.ai/mcp/servers

## Ne İşe Yarar

MCP server'larını aramak, filtrelemek ve karşılaştırmak için kullanıcı arayüzü. GitHub listesinin görsel, interaktif versiyonu. Awesome listesini okumak yerine buradan arama yapmak çok daha hızlı.

## Öne Çıkan Özellikler

- **22,000+ server** — en büyük MCP katalog
- **Günlük güncelleme** — yeni server'lar otomatik ekleniyor
- **Arama ve filtreleme** — kategori, dil, OS, cloud/local
- **Glama Score** — her server için kalite/güven puanı
- **Badge sistemi** — server'lar kendi README'lerine Glama badge ekleyebilir

## Nasıl Kullanılır

```
1. glama.ai/mcp/servers adresine git
2. Arama kutusuna ne aradığını yaz (örn: "github", "slack", "database")
3. Filtreleri kullan (Python/TS/Go, Local/Cloud, macOS vs.)
4. İlgilendiğin server'a tıkla → GitHub reposuna git
5. Repo'daki kurulum talimatlarını uygula
```
