# MapCN

GitHub reposu `AnmolSaini16/mapcn` olan harita bileşeni kütüphanesi.

## Özellikler
- Ücretsiz `MapLibre GL` altyapısı üzerine kurulu.
- `shadcn/ui` ve Tailwind felsefesine uygun, kopyala-yapıştır mimarisi.
- Otomatik light/dark mod, marker, tooltip, rota çizimi.

## Sistemimizdeki Yeri
- UI gerektiren ajan projelerinde (örneğin veri görselleştirme veya dashboard'lar) profesyonel harita arayüzleri oluşturmak için kullanılabilir.
