# Code Wiki

GitHub repolarını analiz edip otomatik dokümantasyon ve mimari diyagramlar çıkaran Google destekli araç (`codewiki.google`).

## Özellikler
- Repo URL'sinden otomatik wiki oluşturma.
- Mimari diyagramları çıkarma.
- Repo'ya özel Gemini chat özelliği.
