# modelcontextprotocol.io — Resmi MCP Sitesi

Model Context Protocol'ün resmi web sitesi ve dokümantasyon merkezi. Anthropic tarafından yönetilir, toplulukla birlikte açık kaynak olarak geliştirilir.

**URL:** https://modelcontextprotocol.io

## MCP Nedir?

AI modellerinin harici araçlara ve veri kaynaklarına standart bir arayüz üzerinden güvenli erişim sağladığı açık protokol. Anlık olmayan veya harici veri gerektiren durumlarda LLM'lerin dış sistemlerle entegre olmasını sağlar.

**Gerçek hayat örnekleri:**
- Claude Code, Figma tasarımından web uygulaması oluşturur
- Agent, Google Calendar ve Notion'a erişerek kişisel asistan olur
- Enterprise chatbot, birden fazla kurumsal veritabanına SQL ile bağlanır
- AI model, Blender'da 3D tasarım yapıp 3D yazıcıya gönderir

## Neden Önemli?

- **Geliştiriciler için:** Her araç için ayrı entegrasyon yazmak yerine tek MCP entegrasyonu
- **AI uygulamaları için:** Hazır araç ekosistemi — veriye ve aksiyona erişim
- **Son kullanıcılar için:** Adına hareket edebilen, gerçek veriyle çalışan daha yetenekli AI

## Geniş Ekosistem Desteği

Claude, ChatGPT, VS Code (Copilot), Cursor, Zed, Sourcegraph Cody, LibreChat, Continue, MCPJam ve daha fazlası
