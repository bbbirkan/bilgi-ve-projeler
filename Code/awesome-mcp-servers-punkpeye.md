# awesome-mcp-servers (punkpeye)

En popüler ve en kapsamlı MCP server koleksiyonu. 2,600+ satır, 22,000+ server'ın Glama diziniyle senkronize. Çoklu dil desteği (Türkçe dahil), aktif Discord topluluğu ve Reddit subreddit'i olan büyük bir awesome listesi.

## Ne İşe Yarar

Model Context Protocol (MCP) ile çalışan server'ların derlenmiş, küratörlü listesi. Claude Desktop, Cursor, Zed ve diğer MCP istemcilerine yükleyebileceğin hazır server'ları bulman için ana referans kaynağı.

**MCP nedir:** AI modellerinin dosya sistemi, veritabanı, API ve diğer servislere standart bir protokolle bağlandığı açık protokol.
